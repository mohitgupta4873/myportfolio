Project 1

Rubik's Cube Solver

In this project, I developed an efficient Rubik's Cube solver using Korf's IDA* algorithm to quickly solve complex configurations. Built in C++ with both 2D and 3D models, the solver leverages data structures to accurately simulate the cube's mechanics. Various search algorithms, including BFS, DFS, and IDDFS, solve 8-move scrambles in under 3 seconds, while Korf's IDA* handles 13-move scrambles in under 10 seconds. This project demonstrates my skills in algorithm design, optimization, and problem-solving, creating a fast and effective solution for Rubik's Cube challenges.

Project 2

Automated Soil Moisture Sensing and Self Watering System

Developed an innovative soil moisture sensor system using Arduino UNO, Arduino Nano, and multiple sensors for accurate soil monitoring. Leveraging C++ and its standard template library, I created an efficient system for precise data collection and analysis. An extensive directory was implemented to log and analyze sensor readings, enabling data-driven decisions for optimal plant care. The system is optimized for energy efficiency by integrating low-power components and sleep modes, making it both sustainable and cost-effective. This project highlights skills in hardware integration, C++ programming, and energy optimization to address environmental and agricultural challenges effectively.
